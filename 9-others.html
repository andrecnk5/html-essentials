Outros

  1. <details> - Define detalhes adicionais que o usuário pode abrir e fechar.
  2. <summary> - Define um cabeçalho visível para um elemento <details>.
  3. <br> - Insere uma quebra de linha.
  4. <abbr> - Define uma abreviação ou um acrônimo.
  5. <mark> - Destaca texto que deve ser realçado.
  6. <nav> - Define um conjunto de links de navegação.
  7. <article> - Define um conteúdo independente.
  8. <section> - Define uma seção genérica de um documento.
  9. <aside> - Define conteúdo tangencial, como barras laterais.
  10. <time> - Define uma hora ou data.
  11. <main> - Define o conteúdo principal do documento.
  12. <figure> - Agrupa conteúdo de mídia com uma legenda opcional.
  13. <figcaption> - Fornece uma legenda para o <figure>.
  14. <picture> - Contêiner para múltiplas fontes de imagem para diferentes resoluções.
  15. <source> - Define uma fonte de mídia alternada para <picture> ou <video>.
  16. <video> - Define um vídeo incorporado.
  17. <audio> - Define um som ou fluxo de áudio.
  18. <embed> - Incorpora conteúdo externo, como plugins.
  19. <object> - Representa um recurso externo, como uma imagem, vídeo ou plugin.
  20. <param> - Define parâmetros para plugins incorporados com <object>.
  21. <canvas> - Usado para desenhar gráficos via scripts.
  22. <noscript> - Define conteúdo alternativo para usuários com scripts desativados.
