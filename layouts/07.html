Formulários e Entradas no HTML

Importância dos Formulários

Formulários são essenciais para a interação do usuário com a página web, permitindo a entrada e envio de dados. Incluem tags como <form>, <input>, <textarea>, <button>, <select>, <option>, <label>, <fieldset>, <legend>, <datalist>, <output>, <progress>, e <meter>.

Exemplo

<form>
  <fieldset>
    <legend>Informações Pessoais</legend>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
    <label for="mensagem">Mensagem:</label>
    <textarea id="mensagem" name="mensagem"></textarea><br>
    <label for="sexo">Sexo:</label>
    <select id="sexo" name="sexo">
      <option value="masculino">Masculino</option>
      <option value="feminino">Feminino</option>
    </select><br>
    <button type="submit">Enviar</button>
  </fieldset>
  <fieldset>
    <legend>Outras Informações</legend>
    <label for="nascimento">Data de Nascimento:</label>
    <input type="date" id="nascimento" name="nascimento"><br>
    <label for="arquivo">Enviar Arquivo:</label>
    <input type="file" id="arquivo" name="arquivo"><br>
    <label for="cor">Escolha uma cor:</label>
    <input type="color" id="cor" name="cor"><br>
    <label for="numero">Número Favorito:</label>
    <input type="number" id="numero" name="numero"><br>
    <label for="range">Escolha um valor:</label>
    <input type="range" id="range" name="range"><br>
    <progress value="70" max="100">70%</progress><br>
    <meter value="2" min="0" max="10">2 de 10</meter><br>
    <label for="pesquisa">Pesquisar:</label>
    <input type="search" id="pesquisa" name="pesquisa"><br>
    <label for="datalist">Escolha uma opção:</label>
    <input list="opcoes" id="datalist" name="datalist">
    <datalist id="opcoes">
      <option value="Opção 1">
      <option value="Opção 2">
      <option value="Opção 3">
    </datalist><br>
    <output name="resultado" for="numero range">Resultado</output>
  </fieldset>
</form>
