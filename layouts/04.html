Importância das Listas

Listas são usadas para agrupar itens relacionados de forma organizada. No HTML, podemos criar listas não ordenadas (<ul>), listas ordenadas (<ol>) e listas de definição (<dl>).

Exemplo

<ul>
  <li>Item de lista não ordenada</li>
  <li>Outro item de lista</li>
</ul>

<ol>
  <li>Item de lista ordenada</li>
  <li>Outro item de lista</li>
</ol>

<dl>
  <dt>Termo de definição</dt>
  <dd>Descrição do termo de definição</dd>
</dl>
