Formulários e Entradas

  1. <form> - Define um formulário para entrada do usuário.
  2. <input> - Define um campo de entrada.
  3. <textarea> - Define uma área de texto.
  4. <button> - Define um botão.
  5. <select> - Define uma lista suspensa.
  6. <option> - Define as opções dentro de uma lista suspensa.
  7. <label> - Define um rótulo para um elemento <input>.
  8. <fieldset> - Agrupa elementos relacionados em um formulário.
  9. <legend> - Define uma legenda para um <fieldset>.
  10. <datalist> - Define uma lista de opções pré-definidas para um campo de entrada.
  11. <output> - Representa o resultado de um cálculo ou ação.
  12. <progress> - Representa o progresso de uma tarefa.
  13. <meter> - Representa uma medida escalar dentro de um intervalo conhecido. 
      
          <!DOCTYPE html>

              <head></head>
