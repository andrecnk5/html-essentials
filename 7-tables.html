Tabelas

  1. <table> - Define uma tabela.
  2. <caption> - Define um título para a tabela.
  3. <thead> - Agrupa o cabeçalho das colunas de uma tabela.
  4. <tbody> - Agrupa o corpo de uma tabela.
  5. <tfoot> - Agrupa o rodapé de uma tabela.
  6. <th> - Define uma célula de cabeçalho em uma tabela.
  7. <tr> - Define uma linha em uma tabela.
  8. <td> - Define uma célula em uma tabela.
  9. <col> - Especifica atributos de coluna.
  10. <colgroup> - Especifica um grupo de uma ou mais colunas na tabela para formatação. 
      
          <!DOCTYPE html>
          <html lang="pt-br">
              <head></head>
              <body>
                  <h3>Tabelas</h3>
                  <table>
                      <caption>Título da Tabela</caption>
                      <colgroup>
                          <col style="background-color:lightgrey">
                          <col style="background-color:white">
                      </colgroup>
                      <thead>
                          <tr>
                              <th>Cabeçalho 1</th>
                              <th>Cabeçalho 2</th>
                          </tr>
                      </thead>
                      <tbody>
                          <tr>
                              <td>Célula 1</td>
                              <td>Célula 2</td>
                          </tr>
                          <tr>
                              <td>Célula 3</td>
                              <td>Célula 4</td>
                          </tr>
                      </tbody>
                      <tfoot>
                          <tr>
                              <td>Rodapé 1</td>
                              <td>Rodapé 2</td>
                          </tr>
                      </tfoot>
                  </table>
              </body>
          </html>
