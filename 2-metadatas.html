Metadados

  1. <title> - Define o título da página.
  2. <meta> - Define metadados sobre o documento que não são representados por outros elementos HTML.
  3. <link> - Usado para linkar folhas de estilo CSS e outros recursos.
  4. <style> - Define estilos CSS diretamente no documento.
  5. <script> - Coloca ou refere a scripts JavaScript.
      
          <!DOCTYPE>
          <html>
              <head>
                <title>Tutorial HTML</title> <!-- Define o título da página -->
                <meta charset="UTF-8"> <!-- Define a codificação de caracteres -->
                <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Ajusta a escala e o tamanho da página para dispositivos móveis -->
                <meta name="keywords" content="HTML, tutorial, aprendizado"> <!-- Palavras-chave para SEO -->
                <meta name="description" content="Tutorial sobre os metadados no HTML."> <!-- Descrição da página para SEO -->
                <meta name="author" content="Seu Nome"> <!-- Nome do autor do documento -->
                <meta name="robots" content="index, follow"> <!-- Instrui motores de busca sobre como indexar a página -->
                <meta http-equiv="X-UA-Compatible" content="IE=edge"> <!-- Garante compatibilidade com versões antigas do IE -->
                <link rel="stylesheet" type="text/css" href="css/styles.css"> <!-- Link para folha de estilo externa -->
                <style></style> <!-- Define estilos CSS diretamente no documento -->
                <script></script> <!-- Coloca ou refere a scripts JavaScript -->
              </head>
              <body></body>
          </html>
